# Test Courses App

Android-приложение для просмотра и управления IT-курсами, разработанное в рамках тестового задания. 

## Функциональные возможности

- **Авторизация**: валидация формата email, проверка заполнения полей, блокировка кнопки входа при некорректных данных. Переход по ссылкам VK и ОК через системный браузер.
- **Главный экран**: загрузка списка курсов из удаленного API, сортировка по дате публикации, статические элементы поиска и фильтрации.
- **Избранное**: добавление и удаление курсов, локальное сохранение через Room, отображение списка избранного.
- **Аккаунт**: экран-заглушка с кнопкой выхода в меню авторизации.
- **Навигация**: постоянное нижнее меню с тремя разделами, сохранение состояния при переключении.

## Технологический стек

- Язык: Kotlin
- UI: Jetpack Compose
- Архитектура: MVVM, Clean Architecture
- DI: Koin
- Сеть: Retrofit, Gson, OkHttp
- Асинхронность: Kotlin Coroutines, Flow
- Локальное хранение: Room
- Загрузка изображений: Coil
- Навигация: Navigation Compose

## Структура проекта

```
core/          # Сетевой слой (Retrofit, DTO), работа с БД (Room, DAO)
domain/        # Бизнес-логика, модели данных, интерфейсы репозиториев
presentation/  # ViewModel и Compose-экраны
di/            # Модули Koin (сеть, база данных, презентация)
ui.theme/      # Цветовые схемы, типографика, темы
```




## Примечания

- Поле поиска и иконка фильтра реализованы как нефункциональные элементы согласно требованиям ТЗ.
- Обрезка описания курсов ограничена двумя строками с использованием `TextOverflow.Ellipsis`.
- Все отступы, размеры шрифтов, скругления и цвета соответствуют исходным макетам.
